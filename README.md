# onestep_imagepicker

https://pub.dev/packages/wechat_assets_picker Customize

### 추가한 기능 
- 첫 GridItem은 사진을 찍을 수 있는 카메라

- 갤러리 내에서 카메라로 사진을 찍고 확인을 누르면 찍은 사진을 자동으로 선택

- 표시되는 앨범명 Recent -> 모든 사진 으로 변경

- 중국어 -> 한국어로 변경
